# Browser Extension

## Installation

To install the browser extension, follow these steps:
1. Download the latest release from the [releases page](https://github.com/Aesir-Technica/word-counter-extension/releases).
2. Unzip the downloaded file.
3. Open your browser's extensions page:
   - For Chrome: `chrome://extensions/`
   - For Firefox: `about:addons`
4. Enable "Developer mode" (Chrome) or click on "Install Add-on From File" (Firefox).
5. Drag and drop the unzipped folder onto the extensions page.

## Usage

Once the extension is installed, you can use it to count words in any text. Simply highlight the text you want to analyze and click on the extension icon in your browser toolbar. The word count will be displayed in a popup.

For more details, check the official documentation or support page.
